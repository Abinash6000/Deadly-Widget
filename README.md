# Deadly Widget

Deadly Widget is a simple yet powerful Android application that keeps you conscious of time by displaying your current age and a live countdown to your next birthday. It features a modern Jetpack Compose UI and a convenient Home Screen widget built with Jetpack Glance.

## Features

- **Live Age Tracking**: See your current age in years.
- **Birthday Countdown**: A real-time countdown (days, hours, minutes, seconds) showing exactly how much time is left until your next birthday.
- **Home Screen Widget**: A glanceable widget that brings the countdown directly to your home screen, updating regularly.
- **Material 3 Design**: Clean, modern interface using the latest Android design standards.
- **Dark Mode Support**: Fully compatible with system-wide dark and light themes.

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **App Widgets**: Jetpack Glance
- **Design System**: Material 3
- **Architecture**: Clean architecture with simple state management.

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio (Ladybug or newer recommended).
3. Build and run the app on an Android device or emulator (API 26+).
4. Enter your Date of Birth in the app.
5. Add the "Age Glance Widget" to your home screen.
